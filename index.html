Recommending Energy-Efficient Java Collections

This page contains pointers to all the primary data we used to conduct our work, please refer to them below.

1. Experiment data

The following archive contains the energy consumption data for all the pairs of device-benchmark we used in our experiments. For each device, there is a folder with its name, and the content of these folders are csv files with the benchmark's name at the beginning and either 'ORIGINAL' or 'MODIFIED' labels at the end of the file name. For our study on energy profiles we included the time spend executing the benchmarks
